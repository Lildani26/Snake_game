# Snake_game
Classic snake game built from scratch using Python and Tkinder. This game includes collision detection, scoring system, random food generation using random and keyboard controls

# Features
 - Check collision
 - Graphical interphase built using Tkinder
 - Smooth movement using window.after() function
 - Real in-game score updates
 - OOP to modeular en clean code (classes Food, and Snake)
